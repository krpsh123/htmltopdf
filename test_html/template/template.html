ASCII stands for American Standard Code for Information Interchange. Computers can only understand numbers, s an ASCII code is the numerical representation of a character such as 'a' or '@' or an action of some sort. ASCII was developed a long time ago and now the non-printing characters are rarely used for their original purpose.
